# 📈 Simple Linear Regression

## 📌 Overview

This project demonstrates the implementation of **Simple Linear Regression**, one of the most fundamental supervised machine learning algorithms used for predicting continuous values.

The notebook covers the complete workflow from loading the dataset to training, evaluating, and visualizing the regression model.

---

## 🎯 Objective

- Understand the concept of Simple Linear Regression.
- Learn how to train a regression model using Scikit-learn.
- Visualize the best-fit regression line.
- Evaluate model performance using regression metrics.

---

## 📂 Dataset

The dataset contains one independent variable (feature) and one dependent variable (target).

Example:

- Study Hours → Exam Score
- Years of Experience → Salary
- Advertising Budget → Sales

---

## 🛠️ Libraries Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📚 Workflow

1. Import required libraries
2. Load the dataset
3. Explore and visualize the data
4. Split the dataset into training and testing sets
5. Train the Simple Linear Regression model
6. Make predictions
7. Visualize the regression line
8. Evaluate model performance

---

## 📊 Evaluation Metrics

The model is evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

---

## 📈 Visualization

The notebook includes:

- Scatter Plot
- Best Fit Regression Line
- Actual vs Predicted Values

---

## 📖 Learning Outcomes

After completing this notebook, you will understand:

- What is Linear Regression?
- Difference between independent and dependent variables
- How the regression line is fitted
- Model prediction process
- Regression evaluation metrics
- Practical implementation using Scikit-learn
