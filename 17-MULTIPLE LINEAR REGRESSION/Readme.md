# 📈 Multiple Linear Regression

This project demonstrates **Multiple Linear Regression**, a supervised machine learning algorithm used to predict a continuous target variable using **multiple independent features**.

- Make predictions on unseen data.

---

## 🧠 What is Multiple Linear Regression?

Multiple Linear Regression models the relationship between a dependent variable and **two or more independent variables**.

The model predicts:

\[
Y = \beta_0 + \beta_1X_1 + \beta_2X_2 + \cdots + \beta_nX_n
\]

where

- **Y** = Target variable
- **β₀** = Intercept
- **β₁...βₙ** = Feature coefficients
- **X₁...Xₙ** = Independent variables

- Sensitive to outliers.
- Multicollinearity can affect performance.
- Requires feature independence assumptions.
