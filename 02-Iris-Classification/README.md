# 🌸 Iris Flower Classification

This project demonstrates a Machine Learning classification model using the famous Iris dataset.

## Objective

Predict the species of an Iris flower based on its measurements.

## Dataset

- Training Dataset: `IRISTRAIN.xlsx`
- Testing Dataset: `IRISTEST.xlsx`

## Machine Learning Workflow

- Load Dataset
- Data Preprocessing
- Train/Test Split
- Model Training
- Model Prediction
- Model Evaluation

## Libraries Used

- Python
- Pandas
- NumPy
- Scikit-learn

## Outcome

Successfully classified Iris flower species using a supervised Machine Learning model.
